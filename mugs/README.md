# mugs/

Drop a square-cropped `<id>.jpg` here and that shill's circle gets a face — no code changes.
The `<id>` is the poster id from `slop-data.js` (e.g. `parscale.jpg`, `kolvet.jpg`).
No image → the map falls back to a styled initials circle, which is always safe.

Sourcing photos of real people is an editorial call: use images you have the right to use
(own screenshots/crops, public-domain, or properly licensed). Keep them square, ~200×200 is plenty.
